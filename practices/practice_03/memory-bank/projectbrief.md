# Project Brief

## Название проекта

**WeatherService** — REST API для уведомлений о погоде

## Контекст

Учебный проект в рамках курса ИТМО «Практика 03: Разработка с помощью AI IDE». Разрабатывается с использованием Roo Code (VS Code) и LLM-агентов.

## Цель

Создать REST API, позволяющий пользователям подписываться на уведомления о погоде для выбранных городов. Данные о погоде получаются через OpenWeatherMap API.

## Требования

### Функциональные
- `GET /weather/{city}` — получить текущую погоду для города
- `POST /subscribe` — подписаться на уведомления (email + город)
- `DELETE /subscribe/{id}` — удалить подписку
- `GET /subscriptions` — список всех подписок

### Нефункциональные
- Хранение данных в PostgreSQL
- Async I/O (FastAPI + asyncpg)
- Валидация входных данных через Pydantic
- Логирование каждого запроса и его исхода
- Конфигурация через переменные окружения

## Стек

Python 3.10+, FastAPI, SQLAlchemy (async), PostgreSQL, httpx, Pydantic v2, Docker

## Область применения

Проект используется на протяжении нескольких практик ИТМО. Артефакты (DoR, DoD, схема БД, тест-план) разрабатывались в рамках Практики 2.
